# Ape Framework starter project (JavaScript)

Install dependencies:

```
npm install
```

Run Ape Framework CLI:

```
npx ape-cli
```

Start API server:

```
npm start
```

Lint code:

```
npm run lint
```

Run tests:

```
npm run test
```

Start API server in production:

```
NODE_ENV=production npx ape-cli api start
```
